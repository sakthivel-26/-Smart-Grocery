
# Smart Grocery Web App

Smart Grocery is a responsive grocery shopping web app built for desktop and mobile users. It is designed to give a simple online grocery experience with clean browsing, quick product discovery, and a smooth checkout-ready UI foundation.

## Features

- Browse grocery items in a modern web interface
- Responsive layout for mobile and desktop
- Fast frontend powered by Vite and React
- Smooth UI interactions with Framer Motion
- Client-side state management using Zustand
- Ready for deployment on Vercel or Netlify

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- Zustand
- Node.js
- Express

## Getting Started

Install dependencies:

```bash
npm install
```

Run the frontend in development:

```bash
npm run dev
```

Run the backend server:

```bash
npm run server
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

```text
src/        Frontend application
public/     Static assets
server/     Express server code
api/        Serverless or API handlers
```

## Deployment

For Vercel:

- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist`

For Netlify:

- Build command: `npm run build`
- Publish directory: `dist`
